## Requisitos Funcionais (RF)

### RF01 — Login do usuário
O sistema deve permitir que o usuário realize login com e-mail e senha.

### RF02 — Recuperar senha
O sistema deve permitir que o usuário solicite recuperação de senha por e-mail.

### RF03 — Atualização de perfil
O usuário pode alterar nome, foto e dados pessoais.

### RF04 — Cadastro de Novo Usuário
O sistema deve permitir que novos usuários criem uma conta fornecendo nome, e-mail válido e uma senha que atenda aos critérios de segurança.

### RF05 — Exclusão de Conta (LGPD)
O sistema deve permitir que o usuário solicite a exclusão definitiva de sua conta e de seus dados pessoais, em conformidade com as normas de privacidade vigentes.
